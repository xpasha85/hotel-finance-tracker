# Hotel Finance Tracker — Decisions

## Роли
- ADMIN: видит всё, включая удалённые, видит историю изменений, может восстановить.
- MANAGER: видит только не удалённые, истории не видит.

## Удаление
Soft delete: Expense.is_deleted=true.
Admin может включать/выключать показ удалённых.

## Деньги
Сумма хранится как amount_cents (integer).
Источник оплаты: CASH / CARD / BANK.

## Фото чеков
1 фото на расход. Хранение локально в папке (volume), без MinIO/S3 на MVP.

## Категории
Пока без вложенности, но схема поддерживает 1 уровень: categories.parent_id.
Архив категории запрещает выбор в новых тратах, но старые траты остаются.
